The cubic polynomial$f(x) = x^3 + ax^2 + b$, where$a$and$b$are non-zero real constants, is tangent to the$x$-axis at a point$x_0 \neq 0$. Which of the following expressions gives the relationship between$a$and$b$?

C: $2a^3 + 27b = 0$
D: $2a^3 - 27b = 0$
E: $27a^3 + 4b = 0$
F: $27a^3 - 4b = 0$

Correct Answer: A

SOLUTION
Reasoning:

For the curve to be tangent to the$x$-axis at$x = x_0$, two conditions must be met simultaneously: 1. The function value is zero:$f(x_0) = 0$. 2. The derivative is zero:$f'(x_0) = 0$. First, we find the derivative:$$f'(x) = 3x^2 + 2ax = x(3x + 2a)$$Setting$f'(x_0) = 0$gives two possible values for$x_0$:$$x_0 = 0 \quad \text{or} \quad x_0 = -\frac{2a}{3}$$Since the question specifies$x_0 \neq 0$, we must have$x_0 = -\frac{2a}{3}$. Next, we substitute this value into the condition$f(x_0) = 0$:$$\left(-\frac{2a}{3}\right)^3 + a\left(-\frac{2a}{3}\right)^2 + b = 0$$$$-\frac{8a^3}{27} + a\left(\frac{4a^2}{9}\right) + b = 0$$$$-\frac{8a^3}{27} + \frac{4a^3}{9} + b = 0$$To combine the fractions, express$\frac{4a^3}{9}$as$\frac{12a^3}{27}$:$$-\frac{8a^3}{27} + \frac{12a^3}{27} + b = 0$$$$\frac{4a^3}{27} + b = 0$$Multiplying through by 27 gives the final relationship:$$4a^3 + 27b = 0$$Hence the correct answer is A.
Key Insight:

"Tangent to the axis" implies the point is a root of both$f(x)$and$f'(x)$; use the simpler linear equation from$f'(x)$to substitute into$f(x)$.
