The tangent to the curve$y = x^3 + k$at the point where$x = 2$passes through the origin. What is the value of the constant$k$?

Correct Answer: F

SOLUTION
Reasoning:

Let$y = f(x) = x^3 + k$. The point of tangency is at$x = 2$. The$y$-coordinate is:$$y(2) = 2^3 + k = 8 + k$$So the tangent touches the curve at$P(2, 8+k)$. Next, we find the gradient of the curve at this point.$$\frac{dy}{dx} = 3x^2$$At$x = 2$, the gradient is$m = 3(2)^2 = 12$. Since the tangent line passes through the origin$(0,0)$and the point$P(2, 8+k)$, its gradient can also be calculated as:$$m = \frac{\Delta y}{\Delta x} = \frac{(8+k) - 0}{2 - 0} = \frac{8+k}{2}$$Equating the two expressions for the gradient:$$\frac{8+k}{2} = 12$$$$8 + k = 24$$$$k = 16$$Alternatively, the equation of the tangent is$y - (8+k) = 12(x - 2)$. Substituting$(0,0)$gives$-(8+k) = -24$, which leads to the same result. Hence the correct answer is 16.
Key Insight:

For a tangent to pass through the origin, the gradient of the chord from the origin ($\frac{y}{x}$) must equal the derivative ($\frac{dy}{dx}$), avoiding the need for the full line equation.
