The function$f$is defined for$0 < x < \pi$by$$f(x) = \frac{\sin x}{1-\cos x} + \frac{\sin x}{1+\cos x}$$What is the minimum value of$f(x)$?

SOLUTION
Reasoning:

We combine the two terms using a common denominator:$$f(x) = \frac{\sin x(1+\cos x) + \sin x(1-\cos x)}{(1-\cos x)(1+\cos x)}$$Expanding the numerator and simplifying the denominator:$$f(x) = \frac{\sin x + \sin x \cos x + \sin x - \sin x \cos x}{1 - \cos^2 x}$$$$f(x) = \frac{2\sin x}{\sin^2 x}$$Since$\sin x \neq 0$for$0 < x < \pi$, we can cancel a factor of$\sin x$:$$f(x) = \frac{2}{\sin x} = 2 \csc x$$The minimum value of$\csc x$on the interval$(0, \pi)$is$1$(occurring at$x = \frac{\pi}{2}$). Therefore, the minimum value of$f(x)$is$2(1) = 2$. Hence the correct answer is C.
Key Insight:

Combining rational trigonometric terms over a common denominator often reveals a single reciprocal function via Pythagorean identities.
