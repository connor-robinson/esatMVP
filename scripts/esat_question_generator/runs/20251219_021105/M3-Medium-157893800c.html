The curve with equation$$y = x^3 - 3ax^2 + b$$where$a$and$b$are non-zero constants, intersects the$x$-axis at exactly two distinct points. Which of the following expresses$b$in terms of$a$?
OPTIONS
A: $b = a^3$
B: $b = 2a^3$

D: $b = -2a^3$
E: $b = -4a^3$

Correct Answer: C

SOLUTION
Reasoning:

For a cubic polynomial to intersect the$x$-axis at exactly two distinct points, one of its stationary points must lie on the$x$-axis (i.e., the local maximum or minimum value must be zero). We differentiate to find the stationary points:$$\frac{dy}{dx} = 3x^2 - 6ax = 3x(x - 2a)$$The stationary points occur at$x = 0$and$x = 2a$. We evaluate the function at these points: 1. At$x = 0$,$y = 0^3 - 3a(0)^2 + b = b$. 2. At$x = 2a$,$y = (2a)^3 - 3a(2a)^2 + b = 8a^3 - 12a^3 + b = b - 4a^3$. The problem states that$b$is non-zero, so the stationary point at$x = 0$cannot be on the$x$-axis. Therefore, the stationary point at$x = 2a$must be the one on the axis. Setting$y(2a) = 0$:$$b - 4a^3 = 0 \implies b = 4a^3$$
Key Insight:

A cubic equation has exactly two distinct roots if and only if one of its local extrema equals zero.
