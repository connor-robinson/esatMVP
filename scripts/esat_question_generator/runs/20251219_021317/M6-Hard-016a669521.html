The function$f$is defined by$$f(x) = \frac{4^x}{4^x + 2}$$What is the value of the sum$$S = \sum_{r=1}^{99} f\left(\frac{r}{100}\right)$$

Correct Answer: B

SOLUTION
Reasoning:

We consider the sum of the function evaluated at$x$and$1-x$:$$f(x) + f(1-x) = \frac{4^x}{4^x + 2} + \frac{4^{1-x}}{4^{1-x} + 2}$$Multiplying the numerator and denominator of the second term by$4^x$:$$\frac{4^{1-x}}{4^{1-x} + 2} = \frac{4^{1-x} \cdot 4^x}{(4^{1-x} + 2) \cdot 4^x} = \frac{4}{4 + 2 \cdot 4^x}$$Dividing the numerator and denominator of this new fraction by 2:$$\frac{2}{2 + 4^x}$$Thus:$$f(x) + f(1-x) = \frac{4^x}{4^x + 2} + \frac{2}{4^x + 2} = \frac{4^x + 2}{4^x + 2} = 1$$The sum consists of terms$f\left(\frac{r}{100}\right)$for$r = 1, 2, \dots, 99$. We can pair terms symmetrically using the property derived above:$$f\left(\frac{r}{100}\right) + f\left(\frac{100-r}{100}\right) = f\left(\frac{r}{100}\right) + f\left(1 - \frac{r}{100}\right) = 1$$The terms run from$r=1$to$r=99$. The pairs are$(1, 99), (2, 98), \dots, (49, 51)$. There are 49 such pairs, contributing$49 \times 1 = 49$to the sum. The remaining middle term is at$r=50$:$$f\left(\frac{50}{100}\right) = f\left(\frac{1}{2}\right) = \frac{4^{1/2}}{4^{1/2} + 2} = \frac{2}{2+2} = \frac{1}{2}$$The total sum is$49 + 0.5 = 49.5$. Hence the correct answer is 49.5.
Key Insight:

Recognising that$f(x) + f(1-x) = \text{constant}$allows the summation to be evaluated by counting pairs rather than calculating individual terms.
