What is the maximum possible value of the function$f(x) = \sin(x) \cos\left(x - \frac{\pi}{4}\right)$as$x$varies over the real numbers?
OPTIONS
A: \frac{2-\sqrt{2}}{4}
B: \frac{1}{2}
C: \frac{\sqrt{2}}{2}
D: \frac{2+\sqrt{2}}{4} âœ“ CORRECT
E: 1

Correct Answer: D

SOLUTION
Reasoning:

We use the product-to-sum identity:$$2\sin A \cos B = \sin(A+B) + \sin(A-B)$$Let$A = x$and$B = x - \frac{\pi}{4}$. Then:$$A+B = 2x - \frac{\pi}{4}, \quad A-B = \frac{\pi}{4}$$Substituting these into the identity gives:$$2f(x) = \sin\left(2x - \frac{\pi}{4}\right) + \sin\left(\frac{\pi}{4}\right)$$$$2f(x) = \sin\left(2x - \frac{\pi}{4}\right) + \frac{\sqrt{2}}{2}$$The maximum value of$\sin\left(2x - \frac{\pi}{4}\right)$is$1$. Therefore, the maximum value of$2f(x)$is:$$2f_{\text{max}} = 1 + \frac{\sqrt{2}}{2} = \frac{2+\sqrt{2}}{2}$$Dividing by 2 gives the maximum value of$f(x)$:$$f_{\text{max}} = \frac{2+\sqrt{2}}{4}$$(Alternatively, using calculus yields the condition$\cos(2x - \frac{\pi}{4}) = 0$, giving$x = \frac{3\pi}{8}$, which requires evaluating$\sin(\frac{3\pi}{8})\cos(\frac{\pi}{8})$via half-angle formulae, confirming the same result.) Hence the correct answer is D.
Key Insight:

Using a product-to-sum identity linearises the expression immediately, separating the variable term from the constant term.
