# Documentation

This directory contains all project documentation organized by purpose.

## Structure

### `/guides`
**Active documentation** for current development and operations:
- Deployment guides
- Security and handover procedures
- Configuration guides (colors, branding)
- Access management
- API documentation

### `/archive`
**Historical documentation** from past development phases:
- Implementation summaries
- Analysis reports
- Completed feature documentation
- Outdated guides and checklists
